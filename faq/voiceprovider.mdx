Enable communication between VAPI AI and the Agency account by configuring your API credentials using the following steps.

**Steps:**

1. Access your newly created **Client** in the agency account

2. Navigate to **Provider Credentials** and select **VAPI **as the voice provider.

3. Enter the API **Private Key** and **Public Key** provided by VAPI.
